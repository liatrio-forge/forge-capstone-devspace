### The old dead end: B can neither push nor pull
machine-b$ devspace workspace push
  Remote branch diverged; run `devspace workspace pull` and reconcile before pushing.  (exit 1)
machine-b$ devspace workspace pull
  Local manifest differs from remote manifest; push or reconcile local changes before pulling.  (exit 1)

### The new path: reconcile (review-first, nothing written yet)
machine-b$ devspace workspace reconcile
Workspace reconcile:
Auto-merged: 1
  ADDED project project_73776ec8bf13
Conflicts: 0
Review saved to DEVSPACE_HOME/last-reconcile.json.

### Reconcile flags the conflict and refuses to apply without an explicit choice
machine-b$ devspace workspace reconcile
Conflicts: 1
  project project_8cc463c33edc field *
    local:  "{... Path:apps/shared ... DefaultBranch:trunk ...}"
    remote: "{... Path:apps/shared ... DefaultBranch:develop ...}"
machine-b$ devspace workspace reconcile --apply
  ERROR  Unresolved reconcile conflicts: - project project_8cc463c33edc ...  (exit 1)

### Explicit resolution: take the remote side
machine-b$ devspace workspace reconcile --force-remote --apply
Auto-merged: 1
  CHANGED project project_8cc463c33edc
Applied reconciled manifest.

### Final state on machine B
  apps/api     type=git  defaultBranch=main
  apps/shared  type=git  defaultBranch=develop
  apps/web     type=git  defaultBranch=main
